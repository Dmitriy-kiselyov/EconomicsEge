− 1-го числа месяца долг увеличивается на r % по сравнению с концом предыдущего месяца, где r – целое число;
− выплата должна производиться один раз в месяц со 2-го по 14-е число каждого месяца;
− 15-го числа каждого месяца долг должен составлять некоторую сумму в соответствии с таблицей: 
Дата	15.01	15.02	15.03	15.04	15.05	15.06	15.07
Долг
(в млн рублей)	1	0,9	0,7	0,5	0,3	0,1	0
Найдите наименьшее значение r, при котором в общей сумме придется выплатить больше 1,4 млн рублей.
